### Time complexity with AVL Tree

|Operation|Time|
|:---:|:---:|
|Search|O(log n)|
|Insert|O(log n)|
|Delete|O(log n)|

- Advantage of using AVL Tree
  - Guarantee: O(log n) for binary search
